# Yahweh Assessment

## Overview

Yahweh Assessment is a React-based application that utilizes various libraries and tools to provide a robust user interface. This project is designed to demonstrate proficiency in React, Material UI, Redux, Formik, and Chart.js, among other technologies.

## Features

- **Material UI**: For modern UI components and styling.
- **Formik and Yup**: For form handling and validation.
- **Chart.js and React-Chartjs-2**: For creating interactive charts.
- **Redux**: For state management.
- **React Router**: For routing and navigation.

## Getting Started
